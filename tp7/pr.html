Abstract

Neste TP foi desenvolvida uma plataforma CRUD (create, read, update, delete) com o objetivo de gerir um arquivo de filmes

Em primeiro lugar, foi necessário instalar o MongoDB, pois neste TP os dados passam a ser guardados numa document store

De seguida, procedeu-se à construção do CRUD, usando o express para facilitar o handle dos pedidos.

Para desenhar o front-end, utilizaram-se templates Pug, que são renderizados no servidor e enviados para o cliente quando é feito um pedido.

Está também incluído um ficheiro com queries Mongo.
