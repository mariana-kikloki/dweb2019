Abstract

Neste trabalho era pedido que se gerasse automaticamente um website para o XML dos arqueossítios. O professor forneceu o código que "parte" o XML em questão em vários ficheiros XML, cada um com um arqueossítio. Providenciou também um XSL que transforma o XML de um arqueossítio numa página HTML.

O XSL é referido no próprio XML, para que, quando o servidor serve o XML pedido, envie também o XSL para que o browser o use como stylesheet, gerando assim a página HTML.
