Abstract

O objetivo deste TP era criar uma lista de tarefas (one-page), que permita adicionar e remover tarefas, ver a lista completa, e deve responder a todos os pedidos

Foi desenvolvido um servidor Node, que responde a todos os pedidos, como requerido, devolvendo páginas de erro, a lista de tarefas ou efetuando operações como adição e remoção de elementos da lista.

Usando templates, foi criado um front-end para apresentar o resultado destes mesmos pedidos.
